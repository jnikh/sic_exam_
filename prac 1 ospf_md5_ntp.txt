B)MD5
*conf t command for router to get in config

router ospf 1
area 0 authentication message-digest

router ospf 2
area 0 authentication message-digest

router ospf 3 
area 0 authentication message-digest

*for R1 in config

interface s0/0/0
ip ospf message-digest-key 1 md5 MD5pa55

*for R2 in config

interface s0/0/0
ip ospf message-digest-key 1 md5 MD5pa55
interface s0/0/1
ip ospf message-digest-key 1 md5 MD5pa55

*for R3 in config

interface s0/0/1
ip ospf message-digest-key 1 md5 MD5p 

*then go to R# using ex and paste

show ip ospf interface
